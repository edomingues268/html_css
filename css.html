CSS

CSS (Cascading Style Sheets) é a linguagem usada para definir a aparencia e o layout das paginas feitas como HTML, controlando cores, fontes, espaçamentos, tamanhos e posicionamento dos elementos; em resumo, enquanto o HTML estrutura o conteúdo, o CSS deixa tudo visualmente organizado e bonito.
